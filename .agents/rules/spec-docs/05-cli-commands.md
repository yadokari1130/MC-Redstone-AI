# 5. CLIコマンド (AI用) 仕様

CLIツールはAIに向けて以下のコマンドを提供する。

## 5.1. `get_blocks`
- **概要**: 指定された座標範囲のブロック状況を取得する。
- **引数** (例):
  - `--x1`, `--y1`, `--z1`: 開始座標 (整数)
  - `--x2`, `--y2`, `--z2`: 終了座標 (整数)
- **標準出力**: 範囲に存在するブロックの状態リスト（JSON文字列）。

## 5.2. `place_blocks`
- **概要**: 考案したレッドストーン回路などの構造物を世界に配置する。
- **引数** (例):
  - `--blocks`: `BlockData`オブジェクトの配列（JSON文字列）、またはJSON配列を含むファイルのパス
- **標準出力**: 配置結果のサマリー（成功数やエラーメッセージなど）。

## 5.3. `interact_block`
- **概要**: 指定した座標のブロック（レバー、ボタン等）を操作して回路を起動・停止する。
- **引数** (例):
  - `--x`, `--y`, `--z`: 対象の座標 (整数)
- **標準出力**: 操作結果（状態がどう変化したか等）。
