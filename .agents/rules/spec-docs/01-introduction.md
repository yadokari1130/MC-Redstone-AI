# 1. はじめに

## 1.1. 背景と目的
本プロジェクトは、Minecraft内でAIが完全に自動でレッドストーン回路を生成・配置・動作検証できるようにするためのCLIツールの開発を目的とする。
従来のコマンド実行による介入では、レッドストーン特有の複雑な状態（BlockState）の正確な取得や、設置順序による回路崩壊の防止が困難であった。
本システムでは、Minecraftサーバー側に専用のHTTPサーバー（プラグイン）を導入し、CLIツールとREST APIで通信する「REST API統一方式」を採用することで、完全自動化・確実な状態取得・安全な一括配置をすべて実現する。

## 1.2. プロジェクトスコープ
本仕様書は以下の2つのコンポーネントの仕様を定義する。
1. **CLIツール**: AIからコマンドを受け取り、Minecraft側のHTTPサーバーへリクエストを送信するGoベースのコマンドラインツール。
2. **Minecraftプラグイン**: Minecraftサーバー内部で軽量なHTTPサーバーを起動し、ブロックの取得・配置・操作を実行するJavaベースのプラグイン。
