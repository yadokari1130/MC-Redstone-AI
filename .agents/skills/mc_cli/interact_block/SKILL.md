# Minecraft ブロック操作 Skill

このスキルは、Minecraft サーバー（Fabric）の HTTP API を使用して、指定した座標にあるブロック（レバー、ボタンなど）に対してインタラクト操作を実行するためのものです。

## 使用方法

`mc-cli` ツールを使用して、以下のコマンドを実行します。

```bash
mc-cli interact-block --pos <座標> [--pos1 <開始座標> --pos2 <終了座標> --delay <待機チック数>]
```

### 引数
- `--pos`: 操作対象のブロックの座標 `"x,y,z"`（整数）。
- `--pos1`: (任意) 操作後に状態を確認したい範囲の開始座標 `"x,y,z"`。
- `--pos2`: (任意) 操作後に状態を確認したい範囲の終了座標 `"x,y,z"`。
- `--delay`: (任意) 操作から状態確認までの待機時間（ゲームチック単位。1チック=50ms）。デフォルトは `0`。
  - インタラクトの結果は即時反映されるものではありません。適切な遅延を挟み、確実に動作が完了した後に状態が取得されるようにしてください。
- `--url`: (任意) サーバーの URL。デフォルトは `http://localhost:8080`。

## 出力形式

コマンドの実行結果は、成功・失敗のステータスを含む JSON 形式で出力されます。
座標（`x1`）が指定された場合、`data` フィールドに指定範囲のブロック情報が含まれます。指定されない場合は、メッセージのみが返されます。

### 成功時のレスポンス例（座標指定なし）
```json
{
  "success": true,
  "message": "ブロックの操作に成功しました"
}
```

### 成功時のレスポンス例（座標指定あり）
```json
{
  "success": true,
  "data": [
    [
      ["minecraft:redstone_wire", [0, 100, 1], {"power": "15"}]
    ]
  ]
}
```

## TIPS
- 回路の動作確認（レバーを引く、ボタンを押す等）に使用します。
- この操作は「右クリック」に相当します。対象がレバーであれば ON/OFF が切り替わり、ボタンであれば一定時間 ON になります。
- `x1` 等の座標と `delay` を併用することで、操作によって回路がどのように変化したかを一度のコマンド実行で確認できます。
- 具体的なブロック情報の形式については `minecraft_get_blocks` スキルを参照してください。
