# Minecraft インベントリ設定 Skill

このスキルは、Minecraft サーバー（Fabric）の HTTP API を使用して、指定した座標にあるブロック（チェスト、ホッパー、ドロッパー、ディスペンサーなど）のインベントリにアイテムをセットするためのものです。

## 使用方法

`mc-cli` ツールを使用して、以下のコマンドを実行します。

```bash
mc-cli set-inventory --pos <座標> --items '<JSON文字列>'
```

### 例: ファイルを指定してセット
```bash
mc-cli set-inventory --pos <座標> --items '@path/to/items.json'
```

### 引数
- `--pos`: 対象ブロックの座標 `"x,y,z"` (整数)。
- `--items`: セットするアイテム情報を含む JSON 配列、または `@` を付けたファイルパス。デフォルトは `[]`（空配列）。

## 入力形式 (JSON)

セットするアイテムのリストを JSON 配列形式で提供します。各要素は `id`（アイテム ID）と `amount`（数量）を持つオブジェクトです。

### 例: チェストへのアイテムセット
```json
[
  {
    "id": "minecraft:redstone",
    "amount": 64
  },
  {
    "id": "minecraft:repeater",
    "amount": 16
  }
]
```

## TIPS
- このコマンドを実行すると、指定した座標のブロックにある**既存のアイテムはすべて消去され**、新しいリストの内容で上書きされます。
- インベントリを完全に空にしたい場合は、`--items '[]'` を指定します。
- 指定した座標にインベントリを持つブロックが存在しない場合、エラーが返されます。
- スロット番号の指定は行えません。提供されたリストの順番に従って、インベントリの先頭スロットから順に配置されます。
