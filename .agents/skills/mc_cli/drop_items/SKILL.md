# Minecraft アイテムドロップ Skill

このスキルは、Minecraft サーバー（Fabric）の HTTP API を使用して、指定した座標にアイテムをエンティティとしてドロップするためのものです。

## 使用方法

`mc-cli` ツールを使用して、以下のコマンドを実行します。

```bash
mc-cli drop-items --x <x> --y <y> --z <z> --items '<JSON文字列>'
```

または、ファイルを指定してドロップします。

```bash
mc-cli drop-items --x <x> --y <y> --z <z> --items '@path/to/items.json'
```

### 引数
- `--x`: アイテムをドロップする X 座標（小数値可）。
- `--y`: アイテムをドロップする Y 座標（小数値可）。
- `--z`: アイテムをドロップする Z 座標（小数値可）。
- `--items`: ドロップするアイテム情報を含む JSON 配列、または `@` を付けたファイルパス。

## 入力形式 (JSON)

ドロップするアイテムのリストを JSON 配列形式で提供します。各要素は `id`（アイテム ID）と `amount`（数量）を持つオブジェクトです。

### 例: アイテムのドロップ
```json
[
  {
    "id": "minecraft:diamond",
    "amount": 1
  },
  {
    "id": "minecraft:iron_ingot",
    "amount": 16
  }
]
```

## TIPS
- アイテムがドロップされると、プレイヤーが拾える状態のエンティティとしてワールドに出現します。
- 座標 (`x`, `y`, `z`) は浮動小数点数を指定できるため、ブロックの中央や特定の高さに微調整してドロップすることが可能です。
  - `x, z`を整数で指定するとブロックの端にドロップしてしまい、他の座標に干渉する場合があります。その場合は0.5を足すことでブロックの中央にドロップさせることができます。
  - `y`を整数で指定するとブロックの上面にドロップしてしまい、他の座標に干渉する場合があります。その場合は0.5を足すことでブロックの中央にドロップさせることができます。
- 重なったアイテムは Minecraft の仕様により自動的にスタックされる場合があります。
