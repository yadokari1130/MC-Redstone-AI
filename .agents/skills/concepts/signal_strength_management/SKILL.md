# 信号強度管理 (Signal Strength Management)

## 概要
レッドストーン信号は0から15の整数値（信号強度）を持ちます。信号強度管理は、この値を意図的に操作・維持することで、特定の距離で信号を途切れさせたり、アナログ的な条件分岐やデータ保持（メモリー）を実現する技術です。

## 仕様・論理
### 1. 信号の減衰 (Signal Decay)
- **レッドストーンダスト**: 信号が1ブロック移動するごとに信号強度は **1減少** します。
- **最大伝達距離**: 強度15の信号は、ダストのみでは15ブロック先で0（無信号）になります。
- **リピーター**: 入力された信号強度に関わらず、出力を **15** にリセット（強化）します。

### 2. コンパレーターによる管理 (Comparator Functions)
信号強度を操作する上で最も重要なコンポーネントです。
- **維持 (Maintenance)**: 背面から入力された信号強度を、そのまま前面に出力します（1ティックの遅延あり）。
- **比較モード (Comparison Mode)**: 背面の入力 $A$ と側面の入力 $B$ を比較し、$A \ge B$ なら $A$ を出力、そうでなければ 0 を出力します。
- **減算モード (Subtraction Mode)**: 右クリックで先端を点灯。式 $A - B$ の結果を出力します（最小値は0）。

### 3. 動力源と信号強度
特定のブロックは、その状態に応じた固有の信号強度を出力します。これを利用して、アナログ回路の定数や変数を生成します。
- **通常のコンテナ**: アイテムの充填率に応じて 0〜15。
- **クラフター (Crafter)**: (アイテム入りスロット + 無効化スロット) 数に応じて **0〜9**。
- **模様入りの本棚 (Chiseled Bookshelf)**: 最後に操作したスロットのインデックスに応じて **1〜6**。
- **書見台 (Lectern)**: 開いているページの位置に応じて 0〜15。
- **大釜 (Cauldron)**: 水位に応じて 0〜3。
- **銅の電球 (Copper Bulb)**: 点灯時に **15**、消灯時に **0**。
- **計算式**: $1 + \lfloor (\text{アイテムの重み合計} / \text{全スロット数}) \times 14 \rfloor$
- これを利用して、任意の強度（1〜15）を自在に作り出すことができます。

## 26.1 における特記事項
- **クラフターのアナログ制御**: スロットの無効化を組み合わせることで、0〜9 の範囲で狙った信号強度を容易に生成できるため、複雑なアナログ演算回路の入力源として非常に優秀です。
- **銅の電球の T-Flip Flop**: 状態を保持しつつ、強度 15 という明確な ON/OFF 信号を出力できるため、アナログ回路のフラグ管理を簡略化できます。

## 使用ガイドライン
- **特定の距離で信号を切る設計**: 特定の距離 $N$ ブロック先でちょうど信号が消えるようにするには、開始地点で強度 $N$ の信号を入力します。
- **アナログ信号の長距離伝送**: 信号強度を維持したまま長距離へ送るには、コンパレーターを並べた「アナログ・リピーター」ライン（$Dust \to Comparator \to Dust$）を使用します。

## 推奨レイアウト
### 簡易減算回路 (Signal Subtractor)
側面から引く値を入力することで、出力を微調整します。
```text
[B: Input Side]
      |
[A: Input Back] -> [Comparator(Subtract)] -> [Output: A-B]
```
