# 強動力と弱動力 (Strong Power & Weak Power)

## 概要
レッドストーン回路において、不透過ブロックが「動力を持っている（出力可能な状態）」には2つの異なる状態があります。これを**強動力**と**弱動力**と呼びます。この違いを理解することは、回路の小型化や信号の干渉（混線）を防ぐために不可欠です。

## 仕様・論理

### 強動力 (Strong Power)
ブロックが、レッドストーンダスト**以外の**部品から直接信号を受けている状態です。
- **入力源**: 
  - レッドストーンリピーター（反復装置）
  - レッドストーンコンパレーター（比較器）
  - レッドストーントーチ（設置されている面から）
  - レバー、ボタン、感圧板、日照センサーなどの入力デバイス
- **出力能力**:
  - 隣接するすべての**レッドストーン装置**（ピストン、ランプ、ドア等）を起動。
  - 隣接するすべての**レッドストーンダスト**に信号を伝える。
  - 隣接するリピーターやコンパレーターを起動。

### 弱動力 (Weak Power)
ブロックが、レッドストーンダストから信号を受けている状態です。
- **入力源**: 
  - ブロックに向かっている、またはブロックの上に乗っているレッドストーンダスト。
- **出力能力**:
  - 隣接するすべての**レッドストーン装置**（ピストン、ランプ、ドア等）を起動。
  - 隣接する**リピーターやコンパレーターを起動**（ここが重要）。
  - 隣接する**レッドストーンダストには信号を伝えない**。

### 比較表
| 特徴 | 弱動力 (Weak Power) | 強動力 (Strong Power) |
| :--- | :--- | :--- |
| **主な入力源** | レッドストーンダスト | リピーター、トーチ、レバー等 |
| **隣接する装置の起動** | ○ | ○ |
| **リピーター等の起動** | ○ | ○ |
| **ダストへの伝達** | **× (不可)** | **○ (可能)** |

## 推奨レイアウト

### 強動力によるダストの起動
リピーターを使ってブロックを強動力状態にすると、その先のダストに信号をリレーできます。
```text
[R] -> [B] -> [D]
[R]: リピーター
[B]: 不透過ブロック (強動力状態)
[D]: レッドストーンダスト (点灯する)
```

### 弱動力による混線防止
ダストでブロックを弱動力状態にすると、隣接する装置（ランプ等）は動きますが、その先のダストには電力が伝わりません。
```text
[D1] -> [B] -> [D2]
[D1]: 入力ダスト
[B]: 不透過ブロック (弱動力状態)
[D2]: レッドストーンダスト (消灯したまま)
```

## 仕様の解説 (26.1対応)
- **銅の電球 (Copper Bulb)**: 
  - 見た目は不透過フルブロックですが、レッドストーン的には **透過ブロック** です。
  - 強動力や弱動力の状態にならず、リピーターやダストからの入力を隣接するブロックやダストへ伝えることはありません。
- **クラフター (Crafter)**: 
  - 見た目通り **不透過ブロック** です。
  - リピーターで強動力化したり、ダストで弱動力化したりすることが可能で、他の不透過ブロックと同様に信号を伝達します。

## 使用ガイドライン
- **信号の延長**: 壁を挟んで信号を伝えたい場合は、リピーターを用いて壁となるブロックを**強動力**にする必要があります。
- **省スペース化**: 隣接した回路で、特定の装置だけを動かしつつ隣の配線に影響を与えたくない場合は、**弱動力**を活用して信号を遮断します。
- **絶縁**: 透過ブロック（ガラス、ハーフブロック、**銅の電球**等）は、強動力・弱動力どちらの状態にもなることができず、信号を一切伝えません。これらを回路の絶縁体として利用できます。
