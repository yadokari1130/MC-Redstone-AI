# 基本論理ゲート (Basic Logic Gates)

## 概要
レッドストーン回路における論理ゲートは、1つ以上の入力信号の組み合わせに基づいて、特定の出力信号を生成する基本的な構成単位です。これにより、条件分岐、メモリ、計算などの複雑なオートメーションが可能になります。
**Minecraft Java Edition 26.1 (1.21+)** では、従来のトーチベースの設計に加え、銅の電球 (Copper Bulb) を用いたよりコンパクトな論理構築が可能です。

## 主要な論理ゲートの仕様・論理

### 1. NOT回路 (Inverter)
- **論理**: 入力がONなら出力はOFF、入力がOFFなら出力はON。
- **入力**: レッドストーン信号（持続的）。
- **出力**: 信号の反転。
- **動作原理**: 不透過ブロックに信号を送り、そのブロックに設置されたレッドストーン信号を消灯させる。

### 2. OR回路 (OR Gate)
- **論理**: いずれか1つ以上の入力がONなら、出力はON。
- **入力**: 2つ以上の入力信号。
- **出力**: 信号の合流。
- **動作原理**: 複数のレッドストーンワイヤーを1つのラインに合流させるだけで成立する。信号の逆流を防ぐには、リピーターを使用する。

### 3. AND回路 (AND Gate)
- **論理**: すべての入力がONのときのみ、出力がON。
- **入力**: 2つ以上の入力信号。
- **出力**: 全入力がONのときのみON。
- **動作原理**: 2つの反転された信号（NOT回路）を合流させ、さらにその合流地点を反転させる（NAND + NOT = AND）。

### 4. XOR回路 (Exclusive OR Gate)
- **論理**: 入力が「異なる」場合のみ ON。両方 ON または両方 OFF なら OFF。
- **構成パターン**:
  - **コンパレーター型 (推奨)**: 信号強度の減算を利用します。
    - 入力 A, B を 3 つのコンパレーター（減算モード）で処理します。
    - `(A - B) + (B - A)` の論理で、両方が同じ強度（0 または 15）なら 0 になります。
  - **銅の電球型**: ボタン等のパルス入力で状態をトグルさせたい場合に有効（T-FF的挙動）。

### 5. その他のゲート
- **NAND回路**: AND回路の出力を反転。
- **NOR回路**: OR回路の出力を反転。トーチを 1 つ置くだけで実現可能。
- **XNOR回路**: XOR回路の出力を反転。入力が「同じ」場合のみ ON。

## 推奨レイアウト

### NOT回路 (標準型)
```text
[入力] (ワイヤー) -> [ブロック] -> [レッドストーントーチ(出力)]
```
- **配置**: ブロックの横または上にトーチを設置。

### AND回路 (標準型)
```text
[入A] -> [ブロA(トーチ)] -- (ワイヤー) -- [出力トーチ]
[入B] -> [ブロB(トーチ)] -------|
```

### XOR回路 (コンパレーター式)
```text
      [B]
       v (側面)
[A] -> [C1] -> [D] -> [出力]
 |      ^       ^
 |    [C2] -----|
 v      ^ (側面)
[A] --- [B]
(C1, C2 は減算モードのコンパレーター, D は合流ダスト)
```
- **配置**: A, B をそれぞれコンパレーターの背面と側面に交差するように入力します。

## 必要な主要ブロック
- `minecraft:redstone_torch`: 信号の反転（NOT）に不可欠。
- `minecraft:redstone_wire`: 信号の伝達と合流。
- `minecraft:repeater`: 信号の延長、逆流防止、遅延制御。
- `minecraft:copper_bulb`: 状態の保持（T-フリップフロップ）やコンパクトな論理構築。
- `minecraft:comparator`: 信号強度の比較や減算、状態の読み取り。

---

## 26.1 における特記事項
- **銅の電球 (Copper Bulb)**: 26.1 においても、1ブロックで T-フリップフロップとして機能する強力なコンポーネントです。2ゲームチック（1レッドストーンチック）の遅延があることに注意してください。
- **ターゲットブロック (Target Block)**: ワイヤーが隣接するブロックに向きを変える性質を利用して、ANDゲートなどの入力をより高密度に配置できます。

## 使用ガイドライン
- **信号の逆流に注意**: OR回路で入力 A の信号が入力 B 側に流れるのを防ぐため、必要に応じてリピーターやコンパレーターで絶縁してください。
- **垂直配置**: ガラスや上ハーフブロックを用いた「レッドストーンラダー」で信号を垂直に伝え、縦型の論理ゲートを構築することで空間を節約できます。
