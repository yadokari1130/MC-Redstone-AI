# 動力源 (Power Sources)

## 概要
レッドストーン回路における動力源は、回路を起動させるためのエネルギーを供給するブロックです。入力の持続性によって「持続信号（Stateful）」と「パルス信号（Momentary）」の2種類に大きく分類され、用途に応じて使い分ける必要があります。

## 仕様・論理

### 信号の種類
- **持続信号 (Stateful/Continuous)**: ON/OFFの状態が保持される信号。レバーやレッドストーンブロックが該当。
- **パルス信号 (Momentary/Pulse)**: 一定時間だけ信号が出力され、自動的にOFFに戻る信号。ボタンや感圧板が該当。

### 主要な入力装置の特性
| ブロック | 入力の種類 | 特性 |
| :--- | :--- | :--- |
| **レバー** | 持続 | 手動で切り替え。ブロックの6面に設置可能。 |
| **ボタン (石類)** | パルス | 10 redstone ticks (1.0秒) 出力。石、磨かれたブラックストーン等。 |
| **ボタン (木類)** | パルス | 15 redstone ticks (1.5秒) 出力。全ての木材、竹。矢やトライデントでも起動。 |
| **レッドストーントーチ** | 持続 | 信号を反転させる（NOTゲート）。1 tickの遅延あり。 |
| **レッドストーンブロック** | 持続 | 常にON。ピストンで動かせるが、信号を切ることはできない。 |
| **感圧板 (石類)** | パルス | プレイヤーとMobのみに反応。石、磨かれたブラックストーン等。 |
| **感圧板 (木類)** | パルス | 全てのエンティティ（アイテム、矢、ボート等）に反応。 |
| **重量感圧板 (金/鉄)** | パルス | 乗っているエンティティの数や総重量に応じて信号強度が変化。 |

## 推奨レイアウト

### 銅の電球による T フリップフロップ (26.1)
銅の電球をコンパレーターで読み取ることで、非常にコンパクトな T フリップフロップを作成できます。

```text
[ボタン] -> [銅の電球] -> [コンパレーター] -> (出力信号)
```
- **利点**: ボタンを押すたびに ON/OFF が切り替わり、状態が保持されます。
- **信号強度**: 点灯時は常に信号強度 15 を出力します。

### 必要な主要ブロック
- `minecraft:lever`
- `minecraft:stone_button`
- `minecraft:oak_button`
- `minecraft:redstone_torch`
- `minecraft:redstone_block`
- `minecraft:copper_bulb`
- `minecraft:comparator`

## 26.1 における特記事項
- **銅の電球 (Copper Bulb)**: 信号を受けるごとに状態が反転する「T フリップフロップ」の性質を持ち、レバーを使わずにボタンで状態を維持したい場合に有効です。2ゲームチック（1レッドストーンチック）の遅延があります。
- **クラフター (Crafter)**: 動力源（ボタン、レバー、パルス信号等）を受けると1回だけクラフト動作を試行します。

## 使用ガイドライン
- **強動力と弱動力**: 
    - 動力源ブロック（レバーやボタン）が設置されているブロックは「強動力」状態になり、全方向に電力を伝えます。
    - レッドストーンダストから電力を受けているだけのブロックは「弱動力」となり、隣接するリピーターなどには伝わりますが、隣のダストには伝わりません。
- **信号の持続時間**: 
    - 石のボタン(1.0s)は短いパルス、木のボタン(1.5s)はやや長いパルスが必要な場合に使い分けます。
