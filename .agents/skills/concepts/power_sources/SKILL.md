# 動力源 (Power Sources)

## 概要
レッドストーン回路における動力源は、回路を起動させるためのエネルギーを供給するブロックです。入力の持続性によって「持続信号（Stateful）」と「パルス信号（Momentary）」の2種類に大きく分類され、用途に応じて使い分ける必要があります。

## 仕様・論理

### 信号の種類
- **持続信号 (Stateful/Continuous)**: ON/OFFの状態が保持される信号。レバーやレッドストーンブロックが該当。
- **パルス信号 (Momentary/Pulse)**: 一定時間だけ信号が出力され、自動的にOFFに戻る信号。ボタンや感圧板が該当。

### 主要な入力装置の特性
| ブロック | 入力の種類 | 特性 |
| :--- | :--- | :--- |
| **レバー** | 持続 | 手動で切り替え。ブロックの6面に設置可能。 |
| **ボタン (石類)** | パルス | 10 redstone ticks (1.0秒) 出力。石、磨かれたブラックストーン等。 |
| **ボタン (木類)** | パルス | 15 redstone ticks (1.5秒) 出力。全ての木材、竹。矢やトライデントでも起動。 |
| **レッドストーントーチ** | 持続 | 信号を反転させる（NOTゲート）。1 tickの遅延あり。 |
| **レッドストーンブロック** | 持続 | 常にON。ピストンで動かせるが、信号を切ることはできない。 |
| **ターゲットブロック** | パルス | 矢やトライデントが当たると ON。中心に近いほど信号強度が強くなる (最大15)。 |
| **日照センサー** | 持続 | 太陽光（または月光）の明るさに応じた信号強度を出力。 |
| **避雷針** | パルス | 落雷時に最大出力 (15) を 8 tick (0.8s) 出力。 |

## 推奨レイアウト

### 銅の電球による T フリップフロップ
銅の電球をコンパレーターで読み取ることで、非常にコンパクトな T フリップフロップを作成できます。

```text
[ボタン] -> [銅の電球] -> [コンパレーター] -> (出力信号)
```
- **利点**: ボタンを押すたびに ON/OFF が切り替わり、状態が保持されます。
- **信号強度**: 点灯時は常に信号強度 15 を出力します。

## 信号の伝達特性 (Strong/Weak Power)

レッドストーン動力源は、周囲のブロックを「動力化 (Powered)」させます。これには 2 つの強さがあります。

- **強動力 (Strong Power)**:
    - **発生源**: レバー、ボタン、レッドストーントーチ、リピーター、コンパレーターの出力。
    - **効果**: 隣接する不透過ブロックを「強動力化」します。強動力化したブロックは、その先のレッドストーンダストやリピーターを起動できます。
- **弱動力 (Weak Power)**:
    - **発生源**: レッドストーンダストからの入力。
    - **効果**: 隣接する不透過ブロックを「弱動力化」します。弱動力化したブロックは、その先の **リピーターやコンパレーター、ピストンなどは起動できますが、レッドストーンダストは起動できません**。

## 使用ガイドライン
- **持続時間**: 石のボタン(1.0s)は短いパルス、木のボタン(1.5s)はやや長いパルスが必要な場合に使い分けます。
- **ターゲットブロックの活用**: 離れた場所から回路を起動したい場合や、信号の向きを変えたい場合にターゲットブロックを配置すると便利です。
