# Minecraft ブロック操作 Skill

このスキルは、Minecraft サーバー（Fabric）の HTTP API を使用して、指定した座標にあるブロック（レバー、ボタンなど）に対してインタラクト操作を実行するためのものです。

## 使用方法

`mc-cli` ツールを使用して、以下のコマンドを実行します。

```bash
./mc-cli/mc-cli interact-block --x <X座標> --y <Y座標> --z <Z座標>
```

### 引数
- `--x, --y, --z`: 操作対象のブロックの座標（整数）。
- `--url`: (任意) サーバーの URL。デフォルトは `http://localhost:8080`。

## 出力形式

コマンドの実行結果は、成功・失敗のステータスを含む JSON 形式で出力されます。

### 成功時のレスポンス例
```json
{
  "success": true,
  "message": "ブロックの操作に成功しました"
}
```

## TIPS
- 回路の動作確認（レバーを引く、ボタンを押す等）に使用します。
- この操作は「右クリック」に相当します。対象がレバーであれば ON/OFF が切り替わり、ボタンであれば一定時間 ON になります。
- 操作の結果として回路がどのように変化したかを確認するには、`minecraft_get_blocks` スキルを併用してください。
