# ホッパー（Hopper）の詳細仕様

## Block ID
- `minecraft:hopper`

## プロパティ (Block States)
- `facing`: ホッパーの排出口が向いている方向を指定します。
  - 値: `down`, `north`, `south`, `west`, `east` （※`up`には設定できません）
- `enabled`: ホッパーがアイテムの移動（搬入・搬出）を行える状態かどうかを示します。
  - 値: `true` (有効), `false` (無効)
  - レッドストーン信号を受けている間は `false` になり、機能が停止します。

## 配置ガイドライン
- `place_blocks` を使用する際、方向（`facing`）を指定する場合は基本的に `attaches` を使用してください。これにより、隣接するブロックに対して正しい向きを自動的に設定できます。ただし、空気ブロックに向かって設置する場合など、アタッチ先のブロックが存在しない場合は、通常通り `facing` プロパティで直接方向を指定してください。
- `down` を指定すると真下のチェスト等へ、水平方向を指定するとその方向にあるコンテナ等へ接続されます。
- 初期状態では `enabled=true` ですが、信号制御を行う場合は回路の入力に合わせて調整します。

## 操作 (Interaction)
- 右クリック（インタラクト）することで、ホッパーのインベントリ（5スロット）を開くことができます。

## 機能・特性
- **アイテムの吸い込み**: ホッパーの真上の空間にあるドロップアイテム、またはすぐ上のコンテナからアイテムを吸い込みます。
- **アイテムの搬送**: 吸い込んだアイテムを `facing` で指定された方向のコンテナへ移動させます。
- **レッドストーンによるロック**: レッドストーン信号を受けている間（`enabled=false`）、アイテムの吸い込みと搬出の両方が停止します。

## 使用方法
- **アイテム輸送**: チェストからチェストへ、あるいは装置間でアイテムを搬送するために使用されます。
- **アイテム仕分け機**: ホッパーの「特定のアイテムだけを吸い込む」特性と、レッドストーンによるロック機能を組み合わせて、自動仕分け機が作られます。
- **バッファ**: 回路の動作中にアイテムを一時的に保持する場所として利用されます。
