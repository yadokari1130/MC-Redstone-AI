# パワードレールの詳細仕様

## Block ID
- `minecraft:powered_rail`

## プロパティ (Block States)
- `powered`: `true`, `false`
  - レッドストーン信号を受けてアクティブな状態（加速状態）であるかどうかを指定します。
- `shape`: `north_south`, `east_west`, `ascending_east`, `ascending_west`, `ascending_north`, `ascending_south`
  - レールの向きと傾斜を指定します。
- `waterlogged`: `true`, `false`
  - ブロック内に水が含まれているかどうかを指定します。

## 配置ガイドライン
- 他のレールと同様に `shape` プロパティで向きを指定します。パワードレールはカーブを作ることができず、直線または傾斜のみ可能です。
- `place_blocks` で設置する際は、隣接するレールとの接続を考慮して `shape` を適切に設定してください。

## 操作 (Interaction)
- プレイヤーによる直接の右クリック操作で状態を変えることはできません。
- レッドストーン信号によって `powered` 状態を制御します。

## 機能・特性
- アクティブ状態 (`powered=true`)：その上を通過するトロッコを加速させます。静止しているトロッコも、背後にブロックがある場合などはその方向に発進させることができます。
- 非アクティブ状態 (`powered=false`)：強力なブレーキとして機能し、通過しようとするトロッコを急停車させます。
- 信号は隣接するパワードレールに最大9ブロックまで伝播します。

## 使用方法
- トロッコの移動速度を維持するための加速路や、目的の場所でトロッコを止めるためのブレーキ、および発射台として使用されます。
- レッドストーン回路と組み合わせることで、特定のタイミングでトロッコを出発させたり停止させたりする自動輸送システムを構築できます。
