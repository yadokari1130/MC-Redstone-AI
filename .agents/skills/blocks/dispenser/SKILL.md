# ディスペンサーの詳細仕様

## Block ID
- `minecraft:dispenser`

## プロパティ (Block States)
- `facing`: ブロックが向いている方向を指定します (`north`, `south`, `east`, `west`, `up`, `down`)。アイテムはこの方向に射出/使用されます。
- `triggered`: レッドストーン信号を受けて作動中かどうかを示します (`true`, `false`)。

## 配置ガイドライン
- `place_blocks` を使用する際、方向（`facing`）を指定する場合は基本的に `attaches` を使用してください。これにより、隣接するブロックに対して正しい向きを自動的に設定できます。ただし、空気ブロックに向かって設置する場合など、アタッチ先のブロックが存在しない場合は、通常通り `facing` プロパティで直接方向を指定してください。射出方向を決定します。
- 矢を射出したり液体を設置したりするため、正面に十分な空間や対象があることを確認してください。

## 操作 (Interaction)
- 右クリック（インタラクト）することで、3x3のインベントリ画面を開くことができます。
- スロットに入れたアイテムの種類によって、作動時の挙動が変化します。

## 機能・特性
- レッドストーン信号の立ち上がりで作動し、インベントリ内のアイテムを1つランダムに「使用」します。主な挙動は以下の通りです：
    - **矢・雪玉・卵:** 飛び道具として射出します。
    - **水/溶岩バケツ:** 正面のブロックに液体を設置、または回収します。
    - **火打石と打ち金:** 正面のブロックに火をつけます。
    - **骨粉:** 正面の植物に対して肥料として使用します。
    - **TNT:** 点火された状態で射出します。
- **準接続 (Quasi-connectivity)**: Java版特有の仕様です。詳細は [quasi_connectivity スキル](../../concepts/quasi_connectivity/SKILL.md) を参照してください。
- **インベントリ操作**: **mc_cli_set_inventory** を使用して直接中身をセットできます。
- **コンパレーター出力**: [inventory_detection スキル](../../concepts/inventory_detection/SKILL.md) を参照してください。

## 使用方法
- 自動防衛システム（矢の連射）や、自動農業（骨粉の使用、収穫用の水流制御）に使用されます。
- レッドストーン回路において、物理的な変化（ブロックの設置やエンティティの生成）を伴う重要な出力装置として機能します。
