# トリップワイヤーフックの詳細仕様

## Block ID
- `minecraft:tripwire_hook`

## プロパティ (Block States)
- `facing`: フックが突き出している方向（`north`, `south`, `east`, `west`）。
- `attached`: フックが糸（Tripwire）によって別のフックと正しく接続され、回路が形成されているかどうか（`true`, `false`）。
- `powered`: エンティティが糸に触れるなどして、レッドストーン信号を出力している状態かどうか（`true`, `false`）。

## 配置ガイドライン
- `facing` プロパティを使用して向きを指定します。フックは不透過フルブロックの側面に設置する必要があります。
- 2つのトリップワイヤーフックを向かい合わせに設置し、その間の床に「糸（`minecraft:tripwire`）」を設置することで回路を形成します。

## 操作 (Interaction)
- 特殊な右クリック操作はありません。

## 機能・特性
- プレイヤー、モブ、アイテム、さらには一部の発射物が糸に触れると作動します。
- 作動時、強度15のレッドストーン信号を出力します。
- フックが取り付けられているブロックを強くパワー（Strong Power）し、フック自体も周囲に信号を伝達します。
- 糸がハサミ以外で壊された場合も信号を出力するため、トラップとしての信頼性が高いです。

## 使用方法
- 通路に設置してプレイヤーやモブの通過を検知するセンサーとして多用されます。
- 糸の長さは最大40ブロックまで延長可能です。
- 感圧版よりも広い範囲、あるいは特定の高さでの検知が必要な場合に適しています。
