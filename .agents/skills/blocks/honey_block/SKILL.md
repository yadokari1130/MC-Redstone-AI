# ハチミツブロックの詳細仕様

## Block ID
- `minecraft:honey_block`

## プロパティ (Block States)
- なし

## 配置ガイドライン
- `place_blocks` を使用して、任意の場所に設置可能です。
- Java版ではレッドストーン信号を遮断（不透明ブロックのような挙動）します。

## 操作 (Interaction)
- 特殊な右クリック操作はありません。
- 側面に密着して落下すると、落下速度が緩和されます（壁擦り）。
- 上に乗っている間はジャンプ力が大幅に制限され、移動速度も低下します。

## 機能・特性
- **接着性**: ピストンによって移動する際、隣接するブロックだけでなく、上に乗っているエンティティ（プレイヤーやアイテム、Mob等）も一緒に移動させます。
- **非接着対象**: スライムブロックとは接着しません。
- **落下ダメージ緩和**: 落下ダメージを80%軽減します。

## 使用方法
- **高度な可動装置**: エンティティを強制的に移動させる特性を利用し、アイテムの輸送路やプレイヤーの移動手段として使用されます。
- **スライムブロックとの併用**: スライムブロックと接着しない特性を活かし、複雑なフライングマシンや隠し扉の機構でパーツを密集させるために使用されます。
