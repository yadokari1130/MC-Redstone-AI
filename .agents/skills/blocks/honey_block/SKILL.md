# ハチミツブロックの詳細仕様

## Block ID
- `minecraft:honey_block`

## プロパティ (Block States)
- なし

## 配置ガイドライン
- `place_blocks` を使用して、任意の場所に設置可能です。
- 技術的には **「透過ブロック」** として扱われます。レッドストーン信号を透過させず、上にダストを置くこともできません。詳細は [block_opacity スキル](../../concepts/block_opacity/SKILL.md) を参照してください。

## 機能・特性
- **接着性**: ピストンによって移動する際、隣接する最大 12 個のブロックを接着します。スライムブロックとは接着しません。詳細は [piston_slime_mechanics スキル](../../concepts/piston_slime_mechanics/SKILL.md) を参照してください。
- **エンティティ運搬**: 上に乗っているエンティティも一緒に移動させることができます。
- **落下ダメージ緩和**: 落下ダメージを80%軽減します。

## 使用方法
- **高度な可動装置**: エンティティを強制的に移動させる特性を利用し、アイテムの輸送路やプレイヤーの移動手段として使用されます。
- **スライムブロックとの併用**: スライムブロックと接着しない特性を活かし、複雑なフライングマシンや隠し扉の機構でパーツを密集させるために使用されます。
