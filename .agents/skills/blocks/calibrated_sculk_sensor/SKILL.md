# 調整されたスカルクセンサーの詳細仕様

## Block ID
- `minecraft:calibrated_sculk_sensor`

## プロパティ (Block States)
- `facing`: `north`, `south`, `east`, `west`（アメジスト入力側の向き）
- `power`: `0`〜`15`（出力されているレッドストーン信号の強度）
- `sculk_sensor_phase`: `inactive`, `active`, `cooldown`（センサーの動作フェーズ）
- `waterlogged`: `true`, `false`（水没状態）

## 配置ガイドライン
- 設置時に `facing` プロパティを使用して、キャリブレーション用信号を受け取るアメジスト側の向きを指定します。

## 操作 (Interaction)
- **キャリブレーション**: アメジストが付いている側面からレッドストーン信号を入力することで、その強度（1〜15）に対応した周波数の振動のみを検知するように設定できます。

## 機能・特性
- **検知範囲の拡大**: 通常のスカルクセンサー（8ブロック）に対し、半径16ブロックまで検知範囲が拡大されています。
- **高速応答**: 作動時間は10ゲームティック（0.5秒）、クールダウンはわずか1ゲームティックと、通常版に比べて非常に高速に動作します。
- **特定周波数のフィルタリング**: キャリブレーション信号を入力していない場合は、通常のスカルクセンサーと同様に全周波数を検知します。

## 使用方法
- **高度なワイヤレス回路**: 特定の行動（例：13＝ブロック設置）にのみ反応する遠隔スイッチなどを、省スペースかつ高速に構築する際に不可欠です。
- **混線防止**: 複数のセンサーが密集する場所で、特定の信号のみに反応させたい場合に活用されます。
