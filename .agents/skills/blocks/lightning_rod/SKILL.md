# 避雷針の詳細仕様

## Block ID
- `minecraft:lightning_rod`

## プロパティ (Block States)
- `facing`: 避雷針の先端が向く方向（`up`, `down`, `north`, `south`, `east`, `west`）。デフォルトは `up`。
- `powered`: 落雷が発生したときに `true` になり、レッドストーン信号が出力されている状態を示します。
- `waterlogged`: 水中に設置されているかどうか（`true`, `false`）。

## 配置ガイドライン
- `facing` プロパティを使用して向きを指定します。
- 設置する際は、`facing` プロパティを使用して先端が向く方向（垂直または水平）を決定します。
- 天井、壁、床のどこにでも設置可能です。

## 操作 (Interaction)
- 斧を使用することで、銅製の場合に錆を削ることができます（このエージェントは主に回路設計を目的とするため、直接的なレッドストーン操作ではありません）。
- ハニカムを使用して錆止め（Waxed）を施すことができます。

## 機能・特性
- 落雷を自身の位置に引き寄せます（半径128ブロック内）。
- 落雷時、強度15のレッドストーン信号を8ゲームティック（0.4秒）出力します。
- 出力中、隣接するすべてのブロックおよび取り付けられているブロックを強くパワー（Strong Power）します。

## 使用方法
- 雷検知センサーとして使用され、天候の変化（雷雨）をレッドストーン信号として取り出すことができます。
- 帯電クリーパーの作成や、銅ブロックの酸化制御、または装飾的なアンテナとして回路の一部に組み込まれます。
