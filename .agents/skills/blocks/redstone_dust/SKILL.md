# レッドストーンダストの詳細仕様

## Block ID
- `minecraft:redstone_wire`

## プロパティ (Block States)
- `power`: `0` 〜 `15` (信号強度。15が最大、0が最小)
- `north`: `none`, `side`, `up` (北側の接続状態)
- `south`: `none`, `side`, `up` (南側の接続状態)
- `east`: `none`, `side`, `up` (東側の接続状態)
- `west`: `none`, `side`, `up` (西側の接続状態)

## 配置ガイドライン
- `place_blocks` スキルを使用して設置します。
- 基本的に `connects` を使用して配置してください。これにより、前後（`from` と `to`）のコンポーネントに基づいた接続状態が適切に計算されます。
- レッドストーンダストは不透明なフルブロック（導電性ブロック）、グロウストーン、逆さのハーフブロックや階段、ガラス、ホッパーの上に設置可能です。
- 空中には設置できません。
- 接続状態（`north`, `south`, `east`, `west`）は周囲のコンポーネントに応じて自動的に更新されますが、`place_blocks` で明示的に指定することも可能です。
- **【重要】ブロックへの接続（動力伝達）の罠**: 
  - ランプやピストンなどのアクチュエータへ信号を伝える際、ワイヤーがそのブロックの方向を向いている必要があります。
  - **解決策**:
    1. **ターゲットブロック (Target Block)**: ワイヤーの隣に `minecraft:target` を置くと、ワイヤーが自動的にその方向を向きます。リピーターを使わずに配線を整理する際に極めて有効です。詳細は [redstone_wire スキル](../../concepts/redstone_wire/SKILL.md) を参照してください。
    2. **直接入力**: ターゲットに向けて「リピーター」または「コンパレーター」を配置する。
    3. **直上配置**: ターゲットブロックの**上**に直接レッドストーンダストを配置する。

## 操作 (Interaction)
- 設置された単一のレッドストーンダストを右クリック（インタラクト）することで、十字（+）形と点（ドット）形を切り替えることができます。
- 点（ドット）形の場合は隣接するブロックに信号を伝えませんが、十字形や通常の配線状態では隣接するブロックや接続されたワイヤーに信号を伝えます。

## 機能・特性
- レッドストーン信号を伝達するメインの導線として機能します。
- 1ブロック伝わるごとに信号強度が1減少します。
- 信号強度が0になると伝達が止まるため、長距離の伝達にはリピーターなどの増幅器が必要です。

## 使用方法
- 回路の配線、信号の伝達および分岐に使用されます。
- ブロックの上面だけでなく、1段の段差であれば昇り降りして接続されます。
