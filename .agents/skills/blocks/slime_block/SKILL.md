# スライムブロックの詳細仕様

## Block ID
- `minecraft:slime_block`

## プロパティ (Block States)
- なし

## 配置ガイドライン
- `place_blocks` を使用して、任意の場所に設置可能です。
- 技術的には「透明（透過）ブロック」として扱われ、レッドストーン信号を透過させ、その上にレッドストーンダストなどを置くことはできません。

## 操作 (Interaction)
- プレイヤーによる右クリック等の直接的なインタラクト操作はありません。
- ブロックの上に乗ると跳ね返り（バウンス）、側面を歩くと移動速度が低下する特性があります。

## 機能・特性
- **接着性**: ピストンによって移動する際、隣接する最大12個のブロックを接着して一緒に移動させます。
- **非接着対象**: 黒曜石やテラコッタ（釉薬瓦）などの不動・難動ブロック、および「ハチミツブロック」とは接着しません。
- **落下ダメージ無効**: 上に着地したエンティティの落下ダメージを無効化します（スニーク中を除く）。

## 使用方法
- **フライングマシン**: ピストンと組み合わせて、ブロックを輸送したり、自走する装置を構築する際の主要なパーツとして使用されます。
- **信号の分離**: ハチミツブロックと隣接させても接着されない特性を利用し、狭いスペースで複数の可動パーツを並行して配置する場合に利用されます。
