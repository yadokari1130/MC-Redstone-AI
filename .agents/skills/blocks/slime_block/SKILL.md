# スライムブロックの詳細仕様

## Block ID
- `minecraft:slime_block`

## プロパティ (Block States)
- なし

## 配置ガイドライン
- `place_blocks` を使用して、任意の場所に設置可能です。
- 技術的には **「透過ブロック」** として扱われます。詳細は [block_opacity スキル](../../concepts/block_opacity/SKILL.md) を参照してください。
- **接着性**: ピストンによって移動する際、隣接する最大 12 個のブロックを接着して一緒に移動させます。ハチミツブロックとは接着しません。詳細は [piston_slime_mechanics スキル](../../concepts/piston_slime_mechanics/SKILL.md) を参照してください。
- **落下ダメージ無効**: 上に着地したエンティティの落下ダメージを無効化します（スニーク中を除く）。

## 使用方法
- **フライングマシン**: ピストンと組み合わせて、ブロックを輸送したり、自走する装置を構築する際の主要なパーツとして使用されます。
- **信号の分離**: ハチミツブロックと隣接させても接着されない特性を利用し、狭いスペースで複数の可動パーツを並行して配置する場合に利用されます。
