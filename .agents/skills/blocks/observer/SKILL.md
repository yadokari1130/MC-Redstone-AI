# 観察者（Observer）の詳細仕様

## Block ID
- `minecraft:observer`

## プロパティ (Block States)
- `facing`: 出力から入力への方向（観察側/顔の側）を指定します。
  - 値: `down`, `up`, `north`, `south`, `west`, `east`
- `powered`: 変化を検知して信号を出力しているかどうかを示します。
  - 値: `true`, `false`

## 配置ガイドライン
- `place_blocks` を使用する際、方向（`facing`）を指定する場合は基本的に `attaches` を使用してください。これにより、隣接するブロックに対して正しい向きを自動的に設定できます。ただし、空気ブロックに向かって設置する場合など、アタッチ先のブロックが存在しない場合は、通常通り `facing` プロパティで直接方向を指定してください。出力から入力への方向（観察したい方向）を指定します。
- 観察側（顔）がターゲットとなるブロックや空間を向き、出力側（赤い点）が反対側を向くように配置されます。
- 例: 上方向を観察し、下方向に信号を出力したい場合は `facing=up` と指定します。

## 操作 (Interaction)
- 直接的な右クリック操作（インタラクト）による状態の変化はありません。
- 隣接するブロックの変化を検知することで動作します。

## 機能・特性
- **ブロック更新の検知**: 正面の1ブロックの範囲内でのブロックの状態変化（設置、破壊、状態変化など）を検知します。
- **信号の出力**: 変化を検知すると、背面から強度15のレッドストーン信号を1レッドストーンチック（2ゲームチック）の間出力します。
- **遅延**: 検知から出力までに1レッドストーンチック（2ゲームチック）の遅延があります。

## 使用方法
- **クロック回路**: 2つの観察者を向き合わせることで、高速なパルスを発生させるクロック回路として使用されます。
- **自動化機構**: サトウキビの成長やカボチャの結実を検知して自動で収穫する機構などに利用されます。
- **フライングマシン**: ピストンと組み合わせて、空中や水中を移動する機構の制御に使用されます。
