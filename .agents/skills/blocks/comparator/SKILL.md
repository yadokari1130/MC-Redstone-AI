# レッドストーンコンパレーターの詳細仕様

## Block ID
- `minecraft:comparator`

## プロパティ (Block States)
- `facing`: `north`, `south`, `east`, `west`
  - **【重要】向きの仕様**: `facing` は **「信号の入力元」** が向いている方角を指します。
  - 例: `facing: "north"` の場合、**北側**から信号を受け取り、**南側**へ向かって信号を出力します。
- `mode`: `compare` (比較モード), `subtract` (減算モード)
- `powered`: `true`, `false` (出力があるかどうか)

## 配置ガイドライン
- `place_blocks` スキルを使用して設置します。
- 不透明なフルブロックや上面が平らなブロックの上に設置可能です。
- **配置時の注意**: 信号を流したい方向（出力先）の方角を `facing` に指定してください。

## 操作 (Interaction)
- レッドストーンコンパレーターを右クリック（インタラクト）することで、「比較モード」と「減算モード」を切り替えることができます。
- 背面の小さなレッドストーントーチが消灯していれば比較モード、点灯していれば減算モードです。

## 機能・特性
- **信号の比較（比較モード）**: 背面入力が側面入力以上であれば、背面入力と同じ強度の信号を出力します。側面入力の方が強い場合、出力は0になります。
- **信号の減算（減算モード）**: 背面入力から側面入力（左右の強い方）を引いた強度の信号を出力します。
- **コンテナ検知**: 背面にチェスト、ホッパー、ディスペンサーなどのインベントリを持つブロックがある場合、その中身の量に応じた信号強度を出力します。
- **遅延**: 信号の変化に対して2ゲームティック（1レッドストーンティック）の遅延があります。

## 使用方法
- 信号強度の計算、アイテム量の検知、クロック回路、論理ゲートの作成などに使用されます。
