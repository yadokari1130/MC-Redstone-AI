# 感圧板の詳細仕様

## Block ID
感圧板には材質によって複数のバリエーションが存在します。

### 木製 (Wooden)
- `minecraft:oak_pressure_plate` (オーク)

### 石製 (Stone)
- `minecraft:stone_pressure_plate` (石)

### 重量感知式 (Weighted)
- `minecraft:light_weighted_pressure_plate` (軽量：金)
- `minecraft:heavy_weighted_pressure_plate` (重量：鉄)

## プロパティ (Block States)
- `powered` (boolean): 木製・石製で使用。ONの時は `true`、OFFの時は `false`。
- `power` (integer, 0-15): 重量感知式で使用。信号強度（0-15）を直接表します。

## 配置ガイドライン
- **設置場所**: ほとんどの不透明ブロック、およびフェンス、壁、ホッパーなどの上面に設置できます。
- **制限**: ブロックの側面や底面には設置できません。
- **支持ブロックの消失**: 下のブロックが破壊されると、感圧板もアイテム化してドロップします。

## 操作 (Interaction)
- 感圧板はプレイヤーによる直接的なインタラクト操作（右クリックなど）で状態を変更することはできません。
- エンティティ（プレイヤー、Mob、アイテムなど）が上に乗ることで起動します。

## 機能・特性
### 起動条件の違い
- **木製**: すべてのエンティティ（プレイヤー、Mob、アイテム、矢、釣り竿の浮きなど）に反応します。
- **石製 / ブラックストーン**: プレイヤーとMobにのみ反応します。（アイテムや矢には反応しません）
- **重量感知式**: すべてのエンティティに反応しますが、乗っているエンティティの数に応じて出力信号強度が変化します。
    - **軽量 (金)**: 1個で強度1、15個以上で最大強度15（1個につき強度1上昇）。
    - **重量 (鉄)**: 10個ごとに強度1上昇（1-10個で強度1、141個以上で最大強度15）。

### 信号出力
- **動力化**: 起動中、感圧板自身と直下の支持ブロックを **強動力 (Strong Power)** します。詳細は [strong_weak_power スキル](../../concepts/strong_weak_power/SKILL.md) を参照してください。
- **信号強度 (重量感知式)**: 乗っているエンティティ数に応じて強度が変化します。詳細は [signal_strength_management スキル](../../concepts/signal_strength_management/SKILL.md) を参照してください。
- **停止遅延**: エンティティが離れてから信号が切れるまで、**10 レッドストーンチック (20 ゲームチック / 1秒)** の遅延があります。

## 使用方法
- **自動ドア**: ドアの前に設置することで、近づくだけで開く自動ドアを作成できます。
- **トラップ / 検知器**: 侵入者の検知や、アイテムがドロップしたことを検知するセンサーとして使用されます。
- **アイテム計数**: 重量感知式を使用することで、ホッパー上のアイテム量などを簡易的に測定できます。
