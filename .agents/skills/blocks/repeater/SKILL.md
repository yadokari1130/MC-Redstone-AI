# レッドストーンリピーターの詳細仕様

## Block ID
- `minecraft:repeater`

## プロパティ (Block States)
- `facing`: `north`, `south`, `east`, `west` (出力方向)
- `delay`: `1`, `2`, `3`, `4` (遅延時間。1〜4レッドストーンティック)
- `powered`: `true`, `false` (出力があるかどうか)
- `locked`: `true`, `false` (信号がロックされているかどうか)

## 配置ガイドライン
- `place_blocks` スキルを使用して設置します。
- 基本的に `connects` を使用して配置してください。これにより、`from` から `to` の方向を向くように `facing` が自動的に計算されます。
- 不透明なフルブロックの上に設置可能です。
- `facing` プロパティを使用して、信号が伝わる方向（出力方向）を指定します。

## 操作 (Interaction)
- レッドストーンリピーターを右クリック（インタラクト）することで、遅延値を `1` から `4` の間で順番に切り替えることができます。
- 遅延値を大きくすると、上のトーチの間隔が広がります。

## 機能・特性
- **信号の増幅**: 入力されたレッドストーン信号を、強度15にリセットして出力します。
- **遅延の付与**: 設定された `delay` 値に応じた時間だけ、信号の伝達を遅らせます。
- **逆流防止**: 信号は背面（入力）から前面（出力）への一方向にしか流れません。
- **信号のロック**: 側面に別のリピーターやコンパレーターから信号が入力されると、現在の出力状態が維持（ロック）されます。

## 使用方法
- 15ブロック以上の長距離配線で信号を再強化する場合や、回路のタイミング調整、ダイオード（逆流防止）として使用されます。
