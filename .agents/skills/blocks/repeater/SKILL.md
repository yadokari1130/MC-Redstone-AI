# レッドストーンリピーターの詳細仕様

## Block ID
- `minecraft:repeater`

## プロパティ (Block States)
- `facing`: `north`, `south`, `east`, `west`
  - **【重要】向きの仕様**: `facing` は**「信号の入力元（背面）」**が向いている方角を指します。信号はその逆の方角へ出力されます。
  - 例: `facing: "north"` の場合、**北側**から信号を受け取り、**南側**へ向かって信号を出力します。
- `delay`: `1`, `2`, `3`, `4` (遅延時間。1〜4レッドストーンティック)
- `powered`: `true`, `false` (出力があるかどうか)
- `locked`: `true`, `false` (信号がロックされているかどうか)

## 配置ガイド
- `place_blocks` (特に `connects`) での設置を推奨。
- **配置時の注意**: 信号を流したい方向（出力したい方向）の**逆**の方角を `facing` に指定してください。

## 特性
- **信号増幅**: 信号強度を15にリセット。
- **遅延**: `delay` プロパティ (1-4 RSチック) で設定。
- **一方向伝達**: 背面から入力、正面から出力。側面からの入力でロック可能。

## 使用方法
- 15ブロック以上の長距離配線で信号を再強化する場合や、回路のタイミング調整、ダイオード（逆流防止）として使用されます。
