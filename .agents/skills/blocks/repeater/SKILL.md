# レッドストーンリピーターの詳細仕様

## Block ID
- `minecraft:repeater`

## プロパティ (Block States)
- `facing`: `north`, `south`, `east`, `west`
  - **【重要】向きの仕様 (Minecraft準拠)**: `facing` は **「信号の出力先（前面）」** が向いている方角を指します。
  - **入力側**: `facing` で指定した方向の **真反対** から信号を受け取ります。
  - **具体例**:
    - `facing: "north"` の場合：**南から入力**を受け取り、**北へ出力**します。
    - `facing: "east"` の場合：**西から入力**を受け取り、**東へ出力**します。
- `delay`: `1`, `2`, `3`, `4` (遅延時間。1〜4 レッドストーンチック / 2〜8 ゲームチック)
- `powered`: `true`, `false` (出力があるかどうか)
- `locked`: `true`, `false` (信号がロックされているかどうか)

## 配置ガイド
- `place_blocks` (特に `connects`) での設置を推奨。
- **配置時の注意**: `facing` には **「信号を送り出したい方向」** を指定してください。
- **配置例**: 東にあるブロックに信号を送りたい場合は、`facing: "east"` と指定します。これにより、入力面（背面）は自動的に西を向きます。

## 特性
- **信号増幅**: 信号強度を15にリセット。
- **遅延**: `delay` プロパティ (1-4 RSチック) で設定。
- **一方向伝達**: 背面から入力、正面から出力。側面からの入力でロック可能。

## 使用方法
- 15ブロック以上の長距離配線で信号を再強化する場合や、回路のタイミング調整、ダイオード（逆流防止）として使用されます。
