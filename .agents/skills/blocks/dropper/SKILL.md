# ドロッパーの詳細仕様

## Block ID
- `minecraft:dropper`

## プロパティ (Block States)
- `facing`: ブロックが向いている方向を指定します (`north`, `south`, `east`, `west`, `up`, `down`)。アイテムはこの方向に排出されます。
- `triggered`: レッドストーン信号を受けて作動中かどうかを示します (`true`, `false`)。

## 配置ガイドライン
- `place_blocks` を使用する際、`facing` プロパティを指定することで排出方向を決定します。
- デフォルトではプレイヤーに向かって設置されますが、回路設計時には意図したコンテナや空間を向くように設定してください。

## 操作 (Interaction)
- 右クリック（インタラクト）することで、3x3のインベントリ画面を開くことができます。
- スロットにアイテムを入れることで、作動時にそれらを排出できるようになります。

## 機能・特性
- レッドストーン信号の立ち上がり（ONになった瞬間）で作動し、インベントリ内のアイテムを1つランダムに排出します。
- 排出先にコンテナ（チェストやホッパーなど）がある場合、その中にアイテムを直接移動させます。
- 排出先にコンテナがない場合、アイテムをエンティティとして世界にドロップします。
- ディスペンサーとは異なり、矢を射出したりバケツを使用したりすることはなく、すべて「アイテム」としてドロップします。

## 使用方法
- アイテムの垂直輸送（ドロッパータワー）や、チェストから別のコンテナへの自動移送に使用されます。
- 1回の信号で1つしか排出しないため、連続して排出させるにはレッドストーンクロック回路が必要です。
