# レッドストーンランプの詳細仕様

## Block ID
- `minecraft:redstone_lamp`

## プロパティ (Block States)
- `lit`: 点灯しているか（`true`/`false`）。

## 配置ガイドライン
- 不透過のフルブロックとして扱われます。
- `place_blocks` を使用して配置する際、初期状態として `lit` を指定できますが、通常は周囲のレッドストーン信号によって制御されます。

## 操作 (Interaction)
- プレイヤーによる直接的なインタラクト操作はありません。

## 機能・特性
- レッドストーン信号（直接または隣接ブロックからの動力）を受け取ると即座に点灯（`lit=true`）します。
- レッドストーン信号が途切れると、2レッドストーンティック（4ゲームティック）の遅延の後に消灯します（Java版特有の挙動）。

## 使用方法
- 照明器具としてだけでなく、レッドストーン回路の出力状態（ON/OFF）を視覚的に確認するためのインジケーターとして広く利用されます。
