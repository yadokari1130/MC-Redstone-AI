# レッドストーンブロックの詳細仕様

## Block ID
- `minecraft:redstone_block`

## プロパティ (Block States)
- なし (プロパティや状態は存在しません)

## 配置ガイドライン
- **向き**: なし。設置時のプレイヤーの向きに影響されず、全方向に対して均一な特性を持ちます。
- **条件**: 不透過ブロックとしての性質を持ち、通常のブロック（石など）と同様に配置可能です。レッドストーントーチとは異なり、支持ブロックを必要としません。

## 操作 (Interaction)
- 特殊な操作（右クリック等）によるインタラクト挙動はありません。

## 機能・特性
- **信号出力**: 隣接する6方向に対して、信号強度15の信号を常時出力します。
- **信号の性質**: 「弱信号源 (Weak power source)」として機能します。隣接するレッドストーンダストやコンポーネントを直接起動させますが、接している不透過ブロックを「動力化」してその先に信号を伝えることはできません。
- **恒久性**: レッドストーン信号をオフにしたり、外部から制御して信号出力を止めることはできません。
- **物理特性**: ピストンや粘着ピストンによって押したり引いたりすることが可能です。

## 使用方法
- **常時オンの信号源**: トーチのように焼き切れることがない、恒久的な信号源として使用されます。
- **移動可能な動力源**: ピストンと組み合わせることで、回路の一部を物理的に動かして信号のON/OFFを切り替えるスイッチのように利用されます。
- **コンパレーター入力**: レッドストーンコンパレーターの側面（比較用入力）や背面（基本入力）に配置するための信号源として使用されます。
