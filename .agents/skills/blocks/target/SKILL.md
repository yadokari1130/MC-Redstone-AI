# 的（ターゲット）ブロックの詳細仕様

## Block ID
- `minecraft:target`

## プロパティ (Block States)
- `power` (integer, 0-15): 出力されているレッドストーン信号の強度。

## 配置ガイドライン
- **向き**: 的ブロックには向き（facing）の概念はありません。どの方向から設置しても同じ見た目と機能を持ちます。
- **場所**: 任意の場所に配置可能です。

## 操作 (Interaction)
- 的ブロックはプレイヤーによる直接的なインタラクト操作（右クリックなど）で状態を変更することはできません。
- 投射物が命中することで起動します。

## 機能・特性
### 信号出力
- **起動条件**: 矢、雪玉、卵、スプラッシュポーション、エンダーパール、三叉戟、ラマの唾液、火の玉などの投射物が命中すると起動します。
- **信号強度**: 命中した位置が中心に近いほど、出力されるレッドストーン信号が強くなります。
    - 中心に命中：強度 15
    - 最も外側に命中：強度 1
- **出力時間**:
    - **矢・三叉戟**: 20ゲームティック（1秒 / 10レッドストーンティック）
    - **その他の投射物**: 8ゲームティック（0.4秒 / 4レッドストーンティック）

### レッドストーンの特性
- **不透明ブロック**: 的ブロックは不透明ブロック（ソリッドブロック）として扱われ、信号を伝達します。
- **ワイヤーの誘引**: 隣接するレッドストーン粉（ワイヤー）を自身の方向へ引き寄せる特性があります。これにより、リピーターなどを使わずにワイヤーを特定のブロックへ向けさせることが可能です。

## 使用方法
- **遠隔スイッチ**: 矢などで撃つことで作動するスイッチとして利用されます。
- **ワイヤーの方向制御**: 回路内でレッドストーン粉の向きを制御し、隣接する他のコンポーネントへの干渉を防いだり、逆に特定の場所へ信号を伝えたりするために使用されます。
- **オブザーバーとの組み合わせ**: 的に命中した際の `power` 状態の変化をオブザーバーで検知して、さらに複雑な回路のトリガーにすることができます。
