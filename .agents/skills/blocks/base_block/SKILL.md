# Minecraft ブロック選択ガイドライン

このスキルは、Minecraft でレッドストーン回路や構造物を構築する際に、どのブロックを使用すべきかの指針を提供します。

## 基本ルール

### 土台および不透過ブロック
- 回路の土台（Foundation）や、信号を伝えるための不透過ブロック（Opaque block）として使用する際は、原則として **`minecraft:iron_block`** (鉄ブロック) を使用してください。

## 理由
- 視認性が高く、回路の構造が分かりやすくなるため。
- 他の素材（石や土など）と区別しやすく、デバッグやメンテナンスが容易になるため。
