# Minecraft ブロック選択ガイドライン

このスキルは、Minecraft でレッドストーン回路や構造物を構築する際に、どのブロックを使用すべきかの指針を提供します。

## 基本ルール

### 土台および不透過ブロック
- 回路の土台や、信号を伝えるための不透過ブロックとして使用する際は、原則として **`minecraft:iron_block`** (鉄ブロック) を使用してください。詳細は [block_opacity スキル](../../concepts/block_opacity/SKILL.md) を参照してください。

### 透過ブロック（配線用）
- 信号を遮断したい場合や、垂直配線（ガラス階段）を作成する際は、**`minecraft:glass`** (ガラス) を使用してください。

## 動力伝達の使い分け
- **強動力 (Strong Power)** を利用して隣接するダストを起動させたい場合は、鉄ブロックなどの不透過ブロックを介して信号を入力します。
- 信号の回り込みを防ぎたい場合は、ガラスなどの透過ブロックを土台にします。詳細は [strong_weak_power スキル](../../concepts/strong_weak_power/SKILL.md) を参照してください。
