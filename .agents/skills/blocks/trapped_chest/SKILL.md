# トラップチェストの詳細仕様

## Block ID
- `minecraft:trapped_chest`

## プロパティ (Block States)
- `facing`: チェストの正面（開閉口）が向く方向（`north`, `south`, `east`, `west`）。
- `type`: チェストの連結状態（`single`, `left`, `right`）。2つ並べてラージチェストにする際に使用されます。
- `waterlogged`: 水中に設置されているかどうか（`true`, `false`）。

## 配置ガイドライン
- 設置する際は、`facing` プロパティを使用して正面（開閉口）の向きを指定します。
- 通常のチェストと同様に、2つ隣接させることでラージ・トラップチェストになります。

## 操作 (Interaction)
- 右クリックでチェストのインベントリを開くことができます。この操作がレッドストーン信号のトリガーとなります。

## 機能・特性
- チェストが開いている間、レッドストーン信号を出力します。
- 出力される信号の強度は、**そのチェストを同時に開いているプレイヤーの数**と等しくなります（最大15）。
- 信号は隣接するブロックに伝わります。特に、チェストの**直下のブロック**や、背面の壁などに信号を伝えるのに適しています。
- 見た目は通常のチェストと酷似していますが、ラッチ（留め金）の周囲がわずかに赤みを帯びているのが特徴です。

## 使用方法
- チェストを開けた瞬間にTNTを爆発させる、あるいは隠し扉を閉めるなどのトラップに使用されます。
- 倉庫などで、誰かがチェストを確認していることを遠隔で検知するシステムにも応用可能です。
- 通常のチェストと隣接させても連結しないため、チェストを隙間なく並べる際にも利用されます。
