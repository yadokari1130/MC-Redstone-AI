# スカルクセンサーの詳細仕様

## Block ID
- `minecraft:sculk_sensor`

## プロパティ (Block States)
- `power`: `0`〜`15`（出力されているレッドストーン信号の強度）
- `sculk_sensor_phase`: `inactive`, `active`, `cooldown`（待機、作動中、クールダウンの各フェーズ）
- `waterlogged`: `true`, `false`（水に浸かっているかどうか。`true`の場合、音を発さなくなる）

## 配置ガイドライン
- ほとんどの不透過ブロックの上に設置可能です。
- 水中に設置して `waterlogged` を `true` にすることで、動作時の音を消すことができます。

## 操作 (Interaction)
- 直接的な右クリック操作による設定変更はありません。
- 羊毛やカーペットを周囲に配置したり、振動源との間に置くことで、特定の方向からの振動を遮断することができます。

## 機能・特性
- **振動検知**: 半径8ブロック以内の範囲で発生した「振動」（歩行、ブロックの設置/破壊、発射装置の動作など）を検知します。
- **作動サイクル**: 振動を検知すると `active` フェーズ（30ゲームティック / 1.5秒）になり、その後 `cooldown` フェーズ（10ゲームティック / 0.5秒）を経て `inactive` に戻ります。
- **信号強度**: 振動源が近いほど強い信号を出力します。式は `max(1, 15 - floor(15 / 8 * distance))` です。
- **周波数**: レッドストーンコンパレーターを接続すると、振動の種類（周波数）に応じた1〜15の信号を出力します。

## 使用方法
- **ワイヤレス通信**: 振動の伝播を利用して、配線なしで信号を伝達する回路に使用されます。
- **プレイヤー検知**: 足音などを検知して自動ドアやトラップを起動させる際に使用されます。
- **特定のアクション検知**: コンパレーターと組み合わせて、特定の行動（食事、着地など）のみに反応する回路を作成できます。
