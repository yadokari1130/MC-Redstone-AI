# 日照センサーの詳細仕様

## Block ID
- `minecraft:daylight_detector`

## プロパティ (Block States)
- `power`: `0`〜`15`（空の明るさに応じたレッドストーン信号の強度）
- `inverted`: `true`, `false`（`true` の場合、反転モード（月光センサー）として動作）

## 配置ガイドライン
- ブロックの高さは 0.375（ハーフブロックより少し低い）です。
- 太陽光（空の明るさ）を検知する必要があるため、上部に不透過ブロックがない場所に設置するのが基本です。

## 操作 (Interaction)
- **mc_cli_interact_block**: 右クリック操作を行い、`inverted` モードを切り替えます。
- モードや明るさに応じて 0 〜 15 のアナログ信号を出力します。詳細は [signal_strength_management スキル](../../concepts/signal_strength_management/SKILL.md) を参照してください。

## 機能・特性
- **光検知**: 空の明るさを検知します。松明などのブロック光（Block Light）には反応しません。
- **通常モード (`inverted=false`)**: 正午に信号強度が最大（15）になり、日の出・日の入りとともに 0 になります。
- **反転モード (`inverted=true`)**: 夜間に信号を出力します。完全に暗い時に最大強度になります。
- **更新間隔**: 20ゲームティック（1秒）ごとに明るさをチェックし、信号強度を更新します。

## 使用方法
- **自動照明システム**: 反転モードを使用して、夜間や天候が悪化した際（雨・雷）にのみ街灯を点灯させる回路に使用されます。
- **時計・タイマー**: 時間帯によって特定の動作（村人の解放、トラップの稼働など）を行わせる際のトリガーとして使用されます。
