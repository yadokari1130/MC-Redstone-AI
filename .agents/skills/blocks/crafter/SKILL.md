# クラフターの詳細仕様

## Block ID
- `minecraft:crafter`

## プロパティ (Block States)
- `orientation`: ブロックの向きを指定します (`north`, `south`, `east`, `west`, `up`, `down`)。完成したアイテムはこの方向の「口」から排出されます。
- `crafting`: クラフト中かどうかを示します (`true`, `false`)。
- `triggered`: レッドストーン信号を受けて作動したかどうかを示します (`true`, `false`)。

## 配置ガイドライン
- `place_blocks` を使用する際、`orientation` プロパティを指定してアイテムの排出方向を決定します。
- 完成品の受け取り用に、正面にチェストやホッパーを配置するのが一般的です。

## 操作 (Interaction)
- 右クリック（インタラクト）することで、3x3のクラフト用インベントリを開きます。
- **スロットの切り替え:** インタラクト画面で各スロットをクリックすることで、そのスロットを**無効化**できます。無効化されたスロットにはホッパーなどでアイテムが搬入されなくなります。これにより、特定の形状のレシピ（例：剣や看板）を自動化することが可能です。

## 機能・特性
- レッドストーン信号の立ち上がりで作動し、現在のグリッド内にあるアイテムを使用してレシピを照合、アイテムをクラフトします。
- 完成したアイテムは即座に正面から排出されます。
- **自動搬入:** ホッパーやドロッパーでアイテムを搬入すると、有効なスロットの左上から順に埋まっていきます。
- **レッドストーン特性:** コンパレータを接続すると、現在「埋まっているスロット（無効化されたスロットを含む）」の数に応じた強度（0〜9）の信号を出力します。これを利用して、材料が揃ったタイミングで自動的にクラフトをトリガーする回路が組めます。

## 使用方法
- サバイバルにおけるアイテム製造の完全自動化に使用されます。
- 無効化スロットとコンパレータを組み合わせることで、複雑なレシピの自動作成ラインを構築できます。
