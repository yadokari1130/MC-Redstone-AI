# レバーの詳細仕様

## Block ID
- `minecraft:lever`

## プロパティ (Block States)
- `face`: レバーが取り付けられている面。
    - `floor`: 床面（上面）
    - `wall`: 壁面（側面）
    - `ceiling`: 天井面（下面）
- `facing`: レバーが向いている方位（Java版では `face` が `floor` または `ceiling` の場合、レバーの軸の向きを示す）。
    - `north`, `south`, `east`, `west`
- `powered`: レバーのON/OFF状態。
    - `true`: ON（信号出力中）
    - `false`: OFF（信号停止中）

## 配置ガイドライン
- **設置場所**: 不透過のフルブロックの上面、側面、下面に設置可能です。また、逆さの階段やハーフブロックの上面にも設置できます。
- 配置は `.agents/skills/mc_cli/place_blocks/SKILL.md` を使用して行います。基本的に `attaches` を使用して配置してください。これにより、土台となるブロックに合わせて `facing` や `face` が自動的に計算されます。

## 操作 (Interaction)
- **切り替え**: プレイヤーがレバーを右クリック（インタラクト）することで、`powered` 状態を `true` と `false` で切り替えることができます。

## 機能・特性
- **電源**: レッドストーン信号の入力源として機能します。
- **信号強度**: ON（`powered=true`）の状態では、レベル15の信号を出力します。
- **動力化**: 
    - 隣接するレッドストーンダスト、コンパレーター、リピーターに信号を伝達します。
    - 取り付けられているブロックを**強動力化（Strong Power）**します。これにより、そのブロックに隣接する他のコンポーネント（ランプやドアなど）を動作させることができます。

## 使用方法
- **回路の入力**: 手動で切り替え可能なスイッチとして、回路の開始地点や特定の機能のON/OFF制御に使用されます。
- **信号のロック**: 定常的なON信号が必要な場所（例えばリピーターのロックや、常にアクティブにしたい回路の一部）に使用されます。
